### Power Pins
- **3.3V** ‚Äî 3.3V output (from onboard regulator, limited current).
- **5V** ‚Äî 5V input (usually from USB or ST-Link).

- **OSC_IN / OSC_OUT** ‚Äî Connects external 8MHz crystal.
- **BOOT0** ‚Äî Selects boot mode (0 = flash, 1 = bootloader).

### Main GPIO Ports

#### **Port A (PA0‚ÄìPA15)**
- **PA0‚ÄìPA7**:

- **PB0‚ÄìPB1**:

  - PWM output

---

#### **Port C (PC13‚ÄìPC15)**
- **PC13‚ÄìPC15**:  
  - Limited GPIO only (weak drive, slow)
